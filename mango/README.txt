Для установки приложение необходимы следующие пакеты:
Node.js не ниже 8-ой версии
npm - пакетный менеджер
mongo база данных

forever (npm install forever) - Пакет для постоянного запуска node.js сервера

Порядок установки:

скопировать содержимое в любую папку, и перейти в папку где находится в файл server.js

Выполнить установку пакетов (npm install) express fs body-parser mango-vpbx mongoose ws

Установить в файлах server.js константы
httpPort - порт на котором будет работать сервер http
wsPort - порт на котором будет работать сервер web socket
login - логин
password - пароль

Установить в файле missed.js констану
wsPort - порт на котором будет работать сервер web socket

проверить что на данных портах не висят разные сервисы и они доступны с внешний стороны

произвести пилотный запуск приложения node server.js

Если все прошло штатно и ошибок при запуске не возникает, прописать а автозагрузке следующую комманду
forever start /путь/server.js
P.S
forever stop /путь/server.js - для остановки сервера
